# WP-03 Stage-2 Evidence Templates

Use these templates to bootstrap Scenario A/B evidence capture immediately after ENG-03 is complete.

## Run Folder Layout

Create:

`scripts/benchmarks/runs/YYYY-MM-DD/<device-id>/`

Then copy these files into that run folder before execution.

## Files

1. `scenario-a.csv`: Scenario A raw rows.
2. `scenario-b.csv`: Scenario B raw rows.
3. `stage-2-threshold-input.csv`: combined A/B rows for threshold evaluation.
4. `notes.md`: run metadata + observations.
