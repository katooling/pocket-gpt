# Operations Framework

This section is the project operating system for cross-functional execution.

## Start Here

1. Open `execution-board.md` to see project-wide status, milestone order, and dependencies.
2. Open your role playbook under `role-playbooks/`.
3. Pick a task marked **Ready** with prerequisites satisfied.
4. Move task status in both places:
   - `execution-board.md`
   - your role playbook
